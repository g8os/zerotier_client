get /network

Get a list of networks this user owns or can view/edit

  • Response

HTTP status code 200

Array of incomplete / preview Network objects

Body

Media type: application/json

Type: array of Network

Items: Network

  • id: (string)

    ZeroTier network ID (16 hex digits)

  • config: (object)
    • nwid: (string)

      16-digit ZeroTier network ID

    • name: (string)

      Short network name, max 255 characters

    • clock: (integer)

      The system clock on the network controller when this network config was last updated (ms since epoch)

    • rules: (array of NetworkRule)

      Items: NetworkRule

      • action: (string)

        currently only 'accept' or 'drop'

      • ruleNo: (integer)

        integer determining rule sort order (ascending in order of ruleNo)

      • etherType: (integer)

        Ethernet frame type (16-bit, higher order bits ignored)

    • relays: (array of NetworkRelay)

      Items: NetworkRelay

      • address: (string)

        10-digit hex ZeroTier address of network preferred relay

      • phyAddress: (string)

        Physical address of relay in IP/port format or null to find using upstream/root

    • private: (boolean)

      If true (default), member access control is enabled.

    • revision: (integer)

      Revision counter (used internally by controller)

    • creationTime: (integer)

      Time network was created on controller microservice (not on ZeroTier Central)

    • v4AssignMode: (string)

      'none' to leave IPv4 addresses alone or 'zt' to have ZeroTier assign and manage them.

    • v6AssignMode: (string)

      'none' to leave IPv6 addresses alone or 'rfc4193' to assign using RFC4193-compliant scheme

    • multicastLimit: (integer)

      Maximum number of recipients for a multicast or broadcast message.

    • ipLocalRoutes: (array of string)

      IP/netmask bits e.g. '10.0.0.0/24'

    • ipAssignmentPools: (array of NetworkIpAssignmentPool)

      Items: NetworkIpAssignmentPool

      • ipRangeStart: (string)

        Start of IP address range (must be within an ipLocalRoute)

      • ipRangeEnd: (string)

        End of IP address range (must be within an ipLocalRoute)

    • controllerInstanceId: (string)

      A random ID representing this instance of the network controller microservice serving this network.

    • authorizedMemberCount: (integer)

      Number of authorized members on this network.

    • memberRevisionCounter: (integer)

      Counter used internally by the network controller microservice to assign member revisions.

  • annot: (object)

    Ephemeral or short-lived internal annotations/flags

  • ui: (object)

    Object for storing arbitrary annotations for UIs (e.g. web UI)
